Proyek ini adalah aplikasi **Employee Management System** yang dikembangkan menggunakan paradigma **Pemrograman Berorientasi Objek (PBO)**.  
Dirancang untuk mempermudah pencatatan, pengelolaan, dan pelaporan data pegawai dalam suatu instansi atau perusahaan kecil.

Dengan aplikasi ini, Anda dapat:
- Menambahkan, mengubah, dan menghapus data pegawai  
- Menampilkan daftar pegawai lengkap dengan atribut masing-masing  
- Melakukan operasi yang menggunakan konsep OOP seperti enkapsulasi, inheritance, dan polymorphism  
- Bisa dikembangkan lebih lanjut sesuai kebutuhan bisnis atau organisasi
